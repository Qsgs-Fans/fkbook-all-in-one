写给玩家
==========

您或许经常听说“新月杀”这个词，但是它的实际项目名字是FreeKill，从下载包的名字也可以看出。
因此本文档中也会尽量只使用FreeKill这一称呼。

FreeKill默认只在客户端中包含了很少的内容。要游玩FreeKill，一般需要加入服务器游玩：

1. 下载FreeKill客户端
2. 点击“加入服务器”
3. 选择想要加入的服务器，输入用户密码后登录
4. 开耍吧

关于上述步骤的详情请看 :doc:`getting-started` 。
